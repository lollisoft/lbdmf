---------------------------------------------------------------- 
*                  Basic development settings                  * 
---------------------------------------------------------------- 
* Please set the following two environment variables to your   * 
* actual unzipped source distribution.                         * 
* As example: If you have unzipped the distribution to C:\,    * 
* thn DEVLW is C: an DEVROOT will be %DEVLW%\dist\develop      * 
*                                                              * 
* The file to be edited is then located at                     * 
* %DEVLW%\dist\develop\projects\cpp\watcomenv.bat              * 
*                                                              *  
* BEFORE you try to compile:                                   *  
*                                                              *  
* Install Open Watcom 1.0 or newer to %DEVROOT%\Tools\Watcom   *  
*                                                              *  
* Install MSVC into %DEVROOT%\Tools\msc                        *  
* DO NOT INSTALL it TO THE DEFAULT LOCATION LIKE:              *  
*                                                              *  
* C:\Programme\Microsoft Visual Studio                         *  
* or                                                           *  
* C:\Program files\Microsoft Visual Studio                     *
